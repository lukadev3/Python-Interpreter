-Pri pisanju koda, koristiti razmak svuda ili ga ne koristiti uopste
-Funkcija print() se koristi samo za stampanje promenljive ili string (print("jabuka") ili print(a))
-Stampanje promenljive se moze izvrsiti i tako sto se u terminalu napise samo naziv promenljive ( a )
-U kodu nisu implementirani excpetion-ovi, jer oni prekidaju program i mora ponovo da se pokrene (kod svakako prepoznaje greske)
-Prilikom dodeljivanja string promenjlivoj mogu se koristiti i ""
i '' (kod automatski postavlja '' kao u pajtonu)
-Pri pokretanju programa kod se moze odmah uneti u terminala a za prekid istog koristiti 'exit'
